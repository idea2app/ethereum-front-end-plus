# MBTI DApp

一个可以查询记录 MBTI 的 DApp。

## 技术栈
- 语言：[TypeScript v5]( https://www.typescriptlang.org/)
- 框架：[Next.js v14](https://nextjs.org/)
- UI 组件库：[Bootstrap v5](https://getbootstrap.com/)

## 使用

```bash
# 安装依赖
pnpm i

# 运行
pnpm dev
```

使用浏览器打开 [http://localhost:3000](http://localhost:3000) 并查看结果。
